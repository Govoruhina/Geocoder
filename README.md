# Геокодер (Россия)

Консольное приложение для геокодирования российских адресов с использованием OSM (Nominatim) и локального кэша в SQLite.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск 

```bash
python main.py
```
```
Или сразу с запросом 
```
```bash
python main.py "Екатеринбург, Родонитовая 1"
python main.py "56.8225650, 60.6177568"
```

## Тесты 

```bash
python -m unittest
```

## Хелп

```bash
python main.py --help
```
